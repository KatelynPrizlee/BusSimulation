Bus Simulation Application

(Mini Project to learn)

Based on observations I made when taking the shuttle from campus to dorm

Front End
>Displays bus and Simulate button, displays the bus after students make their selection

Back End
>Generates a certain number of students 1-48 (the bus seats 48 students, 12 rows 4 columns)

>Randomly generates attributes for each student (Bus stop, Gender)

Technology Used
>Spring Boot
>JS
>CSS
>HTML
